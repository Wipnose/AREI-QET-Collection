# AI Conventions for This Repository

All AI agents interacting with this repository MUST:

1. Read and follow the full specification in:
   `AREI-QET-Residential-OneLine-Rulebook.md`

2. Treat that document as the single source of truth for:
   - graphical style
   - geometry
   - grid rules
   - terminals
   - text rules
   - metadata
   - naming conventions
   - folder structure
   - component families

3. Reject or correct any output that violates the rulebook.

4. Never introduce alternative styles, interpretations, or conventions.

5. Always generate `.elmt` symbols strictly according to the rulebook.
