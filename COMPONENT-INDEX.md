# Component Index — AREI-QET-Collection
Version: 0.1 (initial draft)

This index lists all AREI-style one-line components planned or included in the collection.  
It will evolve based on:
- AREI practice
- NL/FR residential installation standards
- Home Assistant community needs (PV, battery, EV, IoT)
- QElectroTech user contributions

Status codes:
- 🟢 Available
- 🟡 Planned
- 🔵 In progress
- 🔴 Needs review
