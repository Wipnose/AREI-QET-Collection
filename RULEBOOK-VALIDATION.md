# Rulebook Validation Checklist

AI agents MUST validate every generated symbol against:

- [ ] Grid rules
- [ ] Symbol dimensions
- [ ] Origin placement
- [ ] Terminal positions
- [ ] Line style
- [ ] Geometry rules
- [ ] Text rules
- [ ] Layout conventions
- [ ] Metadata rules
- [ ] Naming conventions
- [ ] Folder structure

If any item fails, the output MUST be corrected before use.
